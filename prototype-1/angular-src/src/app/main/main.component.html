Ruka

Hi there, Welcome to Ruka!

The concept is to make a non-verbal way to a product where users could access their smart home system by using a hand gesture. The product will smartly analyze users hand gesture and do certain commands that are set beforehand.
How to use Ruka

Use 'Five' Hand Gesture to Start and to End the command

Between start and end, there will be 2 hand gestures you could do in order to run a command.
Two Hand Gestures

( 4, 0 ) Show the news
( 4, 1 ) Show entertainment news
( 4, 2 ) Show technology news
( 3, 0 ) Show the weather now
(cont. ) Two Hand Gestures

( 2, 0 ) Make an alarm tomorrow at 6
( 1, 0 ) Turn off all lights
( 1, 2 ) Turn on all lights
( 3, 2 ) Show the notification
